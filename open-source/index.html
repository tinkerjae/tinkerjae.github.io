Open Source + Attributions


Categories: misc;

This page is mainly for open source accreditation and attribution.

Website Info

My website is open source and the source code is public @ the previous link. My website is running Jekyll 4.4.1, with the jekyll-feed and jekyll-paginate-v2 plugins being used for Atom/RSS feed generation, as well as pagination + the autopages feature for categories and tags. My website is hosted through Github Pages with a custom domain via Cloudflare. All of my photos, images, and non embeded media are hosted at Catbox.moe.

Development Process

My dev process is different than others because I’m running plugins that GH Pages doesn’t support natively. I edit/modify my source branch to modify the website in any capacity, but the changes only show once I upload the raw html, css, and other ‘_site’ files to the static-files branch. The static-files are what actually show up on the website where as the source files exist on the backend.
